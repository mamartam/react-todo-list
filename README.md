## 🔌 About Project
This project is based on my previous assignment—a Todo List built with Vanilla JS. This time, I rebuilt the project using **React** and bundled it with **Vite**. Using Vite is incredibly convenient because it is super fast and provides a great basic project structure right out of the box.
In this project, I used:
* `JSX` for the UI structure
* React Hooks: `useState` and `useEffect`
* `Props` for passing data between components

## 👩🏻‍🎓 What I Learned

- Project Configuration: How to configure the project by installing npm packages (like Sass for styling).
- React Hooks & Props: How to use hooks properly and apply props in practice to pass data.
- Component Architecture: How to split code into components and import them from different files into the main App component.
- Forms in React: How to handle forms and manage inputs using `useState`.
- Deployment: How to deploy React applications bundled with Vite to GitHub Pages.

## 🚀 How It Can Be Improved?

While the app is fully functional, here are a few features I would like to add in the future to make it even better:

- Edit Task Feature: Add the ability to edit the text of an existing todo item instead of just deleting it.
- Animation Effects: Implement smooth transitions or animations when adding, completing, or deleting tasks (using CSS or libraries like Framer Motion).
- Clear All Button: Add a button to quickly clear all completed tasks or wipe the entire list at once.
- Task Categories/Tags: Allow users to categorize tasks (e.g., Work, Personal, Shopping) with different color tags.
- Due Dates: Add a calendar picker to set deadlines for tasks and show a visual warning if a task is overdue.
